# Sentiment Analysis using Naive Bayes

## About this
This was part of my text processing / NLP coursework.

I implemented a basic sentiment analysis system using:
- Naive Bayes
- Rule-based classification

---

## What the code does

The model:
- reads movie reviews dataset
- splits into training and testing
- calculates probabilities of words
- predicts whether a sentence is positive or negative

---

## Extra logic added
I also added:
- handling of negation words (like "not")
- intensifiers (like "very", "extremely")
- simple scoring system for rule-based classification

---

## Metrics calculated
- Accuracy
- Precision
- Recall
- F1 Score

---

## What I learned
- how Naive Bayes works practically
- why preprocessing matters
- difference between rule-based vs ML models
